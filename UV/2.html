• CFX UVCal Routine Input File Generator for V17.2

  • [ 간단 사용법 ]

    1. (필수사항) Input File 이름을 적절히 입력
    2. (선택사항) Input File 이 저장될 위치를 입력
    3. 각각의 항목에 맞는 파라미터를 입력합니다.
    - 일반적으로 Reactor Parameter는 Number of Lamp 와 Water UVT 만 수정하면 됩니다.
    4. 램프관련 파라미터를 아래의 항목에 입력(중압램프는 두번째 시트도 입력)
    5. 입력 완료 후 , "Input File Generation" 을 더블클릭 하면, 앞서 설정한 위치에 입력파일이 생성됩니다.
    6. 생성된 입력파일을 CFX Pre 에서 적절히 선택하여 줍니다.
